O que é o HTML5?

O HTML5 é uma linguagem de marcação de texto maioritatiamente utilizada para a criação de websites. Basicamente é a estrutura de um website. A maior parte dos websites que acessamos, são feitos em HTML: Google, YouTube, Facebook, etc...

O HTML (HyperText Markup Language) foi criado no início dos anos 90 por Tim Berners-Lee, cientista da computação britânico e inventor da World Wide Web (WWW). Ele precisava de uma forma de organizar e conectar documentos através de hiperlinks na internet, permitindo que pesquisadores compartilhassem informações facilmente.

Diferença entre HTML5, CSS3 e JavaScript

Se conhece HTML então provavelmente já ouvir falar dos termos CSS e JavaScript. Vamos aprender a diferença entre estas 3 tecnologias agora mesmo.

HTML5 é usada para marcação de texto. É basicamente o esqueleto da página. Com ele podemos introduzir textos, imagens, videos, formulários, etc.

CSS3 é usada para estilizar o conteúdo. É a linguagem de estilo de caixa do HTML5. Com ele podemos mudar cores, tamanhos, fontes, posições, etc.

JavaScript é usado para criar interatividade e manipular o conteúdo da página. É a linguagem que permite fazer a interação com o utilizador, como cliques, scrolls, etc.

Resumidamente: HTML é o esqueleto, o CSS é a pele e o JavaScript é o cérebro.

Instalar as ferramentas necessárias.

Para começar a trabalhar com HTML5, você precisa dos seguintes materiais:

  • Um editor de texto: Usaremos o Visual Studio Code
  • Um navegador web: Aconselhamos a usar o Chrome ou Firefox
  • Conexão à internet estável: Afinal de contas, estamos a trabalhar na web.

Agora que já temos as ferramentas necessárias, estamos quase prontos para começar a trabalhar. Na próxima aula iremos mexer com o nosso código pela primeira vez.
