Criar um ficheiro HTML

Para criarmos um ficheiro HTML, precisamos abrir o Visual Studio Code e selecionar a opção "New File".

Depois de clicarmos em "New File" vai aparecer uma janela no topo do Visual Studio Code. Devemos digitar o nome do ficheiro com a extensão ".html" e clicar na tecla ENTER.

Depois disso, surgirá outra janela para confirmarmos o nome do ficheiro a ser criado e o diretório onde será armazenado. Depois de confirmar estas duas informações, clique em "Create file"

Feito isso, já teremos o nosso primeiro ficheiro HTML criado. A interface pode não ser a mesma que a da imagem devido à algumas extensões que já estão instaladas na nossa máquina. Mas não se preocupe, estás no caminho certo.

Montar a estrutura básica do HTML5

Para criarmos a estrutura base do HTML5, vamos utilizar a lista de atalhos do VS Code chamada "Emmet Abraviation". Basta inserir o sinal de exclamação "!" e clicar em ENTER

Agora vamos aprender o que são tags e para quê que elas servem em HTML.

Tags

Uma estrutura HTML é formada por Tags. Tags são componentes que permitem inserir informações dentro do nosso ficheiro HTML como títulos, parágrafos, imagens, etc...

Quando criamos a estrutura base HTML, nos deparamos com várias tags. Vamos aprender para que servem cada uma delas

  • <!DOCTYPE html> - Tag de declaração de tipo de documento HTML5. É necessária para usar todas as novas tags HTML5.
  • <html> - Tag de abertura do documento HTML5.
  • <head> - Tag que contém que contém informações importantes sobre a página.
  • <meta> - Tag usada para definir meta-dados dentro do head como o conjunto de caracteres, largura da página, etc.
  • <title> - Tag usada para definir o título da página.
  • <body> - Tag que contém o conteúdo principal e visível da página.

Repare que algumas tags como html precisam de ser abertas e fechadas, mas outras tags como meta são apenas de fechamente. Isso acontece porque as tags de abertura e fechamento precisam receber tags filhas dentro delas, já as tags de apenas fechamento só recebem atributos. Não precisas de decorar quais são as tags de abertura e fechamento ou as tags de apenas fechamento (simples), ao longo do curso vais pegar o jeito mas caso tenhas alguma dúvida podes sempre consultas todas as tags em W3SChools
