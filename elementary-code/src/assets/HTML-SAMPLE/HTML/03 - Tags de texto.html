Live Server

Melhor do que escrever código é ver o mesmo a funcionar. E para vermos o nosso código em ação no nosso navegador, vamos instalar a extensão Live Server no VS Code

Depois de instalar a extensão basta clicar no botão "Go Live" no rodapé para abrir o ficheiro HTML no navegador e ter uma visão de como o site está.

Títulos

Os títulos em HTML são introduzidos pelas tags <>. Eles possuem uma hierarquia de 1 a 6, onde quanto menos for o número maior é a sua importância.

Parágrafos

Os parágrafos são introduzidos pela tag <p> e terminados pela tag </p>.

Textos em negrito e itálico

Para destacarmos um texto a negrito colocamos entre as tags <b> </b>. E para destacarmos um texto em itálico usamos <i> </i>

Quebra de linha

Se reparares bem, apesar de haver uma quebra de texto no parágrafo no editor de texto, no navegador não mudou nada. Isso é porque o ENTER serve apenas para organizar o texto no editor de texto. Caso queiramos quebrar uma linha usamos a tag </br>

Agora já sabemos como inserir títulos, paragráfos, como colocar textos em negrito e itálico e por fim aprendemos a quebrar linhas.
